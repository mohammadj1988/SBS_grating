# SBS Phonon FDTD

A 2D finite-difference time-domain (FDTD) model of the acoustic phonon field generated by
stimulated Brillouin scattering (SBS) in a co-propagating pump/probe geometry. It solves for the
elastic wave launched by two crossed optical beams and checks the result against the analytic
Brillouin frequency relation.

## Physics background

When two coherent optical beams (pump + probe) cross at an angle `theta`, their interference
pattern creates a moving intensity grating. Through electrostriction this grating drives a
travelling acoustic (phonon) wave with wavevector set by phase matching:

```
q = 2 k_opt sin(theta / 2)
```

The resulting phonon frequency is the Brillouin frequency:

```
f_B = (2 n v / lambda_opt) sin(theta / 2)
```

where `n` is the refractive index, `v` the relevant elastic wave speed (longitudinal or shear),
and `lambda_opt` the optical wavelength. This code solves the full 2D elastic wave equation for
that forced response (rather than just plugging into the formula) and confirms the FDTD result
matches the analytic prediction.

## What the code does

- `materials.py` — material properties (refractive index, longitudinal/shear wave speed, density)
  for a couple of reference media (e.g. water, collagen gel).
- `config.py` — a single `SimConfig` dataclass holding every tunable parameter (wavelength, angle,
  NA, domain size, resolution, frequency sweep range, damping, etc.), loadable from a YAML file
  and overridable from the CLI.
- `forcing.py` — builds the Gaussian pump-probe interference envelope that drives the body force
  term in the elastic wave equation.
- `fdtd.py` — the 2D elastic FDTD solver itself (`ElasticFDTD2D`), including an absorbing sponge
  layer at the domain boundary.
- `studies.py` — three parameter studies:
  - **Study A**: frequency sweep at fixed pump angle, to locate the resonant Brillouin peak.
  - **Study B**: sweep of pump ring half-angle (set by the objective NA) to trace how `f_B` shifts
    with angle.
  - **Study C**: time-domain snapshots of the launched phonon wavefront (longitudinal and shear).
- `plotting.py` — renders the sweep results and time-domain snapshots as static plots/animations.
- `validation.py` — automated sanity checks comparing the FDTD output against theory:
  - numeric Brillouin peak vs. the analytic `f_B` formula,
  - shear response is negligible in a zero-shear-modulus medium (water) vs. present in a
    viscoelastic one (collagen gel),
  - the absorbing boundary doesn't let energy reflect back into the domain.
- `main.py` — CLI entry point that wires all of the above together.

## Usage

From the repository root:

```bash
python -m sbs_phonon_fem.main --studies all
python -m sbs_phonon_fem.main --quick --studies a
python -m sbs_phonon_fem.main --config sbs_phonon_fem/config.yaml --material collagen_gel
```

`--quick` uses coarser/faster settings for smoke-testing the pipeline rather than production
quality results. Output plots, animations, and a validation report are written to `outputs/`
(or the directory given by `--out`).

## Validation

The last full run passed all automated checks (`outputs/validation_report.txt`):

- Numeric vs. analytic Brillouin peak: 0.3% relative error (well within the 15% tolerance, which
  is loose because the sweep grid and finite forcing-spot bandwidth broaden the peak).
- Shear response in water is negligible compared to collagen gel (ratio 0.117, threshold 0.2).
- Absorbing boundary shows no meaningful energy rebound (2.0%, threshold 5%).
